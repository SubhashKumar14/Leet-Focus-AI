# 🚀 Quick Installation Guide

## Load Chrome Extension

1. **Open Chrome Extensions Page**

   ```
   Type in address bar: chrome://extensions/
   ```

2. **Enable Developer Mode**

   - Look for "Developer mode" toggle in top-right corner
   - Click to enable it

3. **Load the Extension**

   - Click "Load unpacked" button
   - Navigate to and select the `leetcode-focus-extension` folder
   - Click "Select Folder"

4. **Verify Installation**
   - Extension should appear in your extensions list
   - You'll see "LeetCode Focus - AI Assistant & Focus Mode"
   - Status should show "Enabled"

## Test the Extension

1. **Go to LeetCode**

   - Visit: https://leetcode.com/problems/two-sum/

2. **Click Extension Icon**

   - Look for the extension icon in Chrome toolbar
   - If not visible, click the puzzle piece icon and pin it

3. **Enjoy the Features!**
   - 🤖 AI Assistant for hints and code improvements
   - 🎯 Focus Mode to hide distractions
   - 🗺️ Learning Roadmaps for structured practice

## Troubleshooting

- **Extension not visible**: Check if it's in the extensions menu (puzzle piece icon)
- **Not working on LeetCode**: Make sure you're on a problem page
- **Permission denied**: Refresh the LeetCode page after loading extension

## Ready to Code! 🎯

Your LeetCode Focus extension is now ready to enhance your coding practice experience!
